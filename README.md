
# Narmada Upwan and Sanshodhan Kendra

## Run locally
```bash
npm install
npm run dev
```

## Deploy on Vercel
1. Push this folder to GitHub.
2. Import the repo into Vercel.
3. Add your domain `narmadaupwan.org`.
4. Replace placeholder images in `/public` with your final assets.

## Important files
- `app/page.jsx`
- `app/layout.jsx`
- `app/globals.css`
- `public/logo.png`
- `public/favicon.png`
- `public/gallery1.jpg` to `public/gallery6.jpg`
